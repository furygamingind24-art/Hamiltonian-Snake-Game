# Hamiltonian Snake Game

An AI-powered Snake Game developed in Python using Pygame and NumPy.

## Features
- Hamiltonian Cycle Pathfinding
- Autonomous Snake Navigation
- Dynamic Speed Control
- Full Board Completion Logic
- Real-time Statistics Dashboard

## Technologies Used
- Python
- Pygame
- NumPy

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the game:
   python snake_game.py

## Controls
- SPACE: Toggle AI / Reset after win
- UP ARROW: Increase speed
- DOWN ARROW: Decrease speed
